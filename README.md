# Leave Management API 

# Project Overview
This is a Leave Management System built using FastAPI.  
It allows employees to apply for leave and managers to approve or reject requests.

# Tech Stack
- FastAPI
- Uvicorn
- SQLite
- SQLAlchemy
- Pydantic
- JWT Authentication
  
# Features
- User Signup & Login
- JWT Authentication
- Apply Leave
- Approve / Reject Leave
- Filter Leaves
- Pagination
- Logging
- Health Check Endpoint
